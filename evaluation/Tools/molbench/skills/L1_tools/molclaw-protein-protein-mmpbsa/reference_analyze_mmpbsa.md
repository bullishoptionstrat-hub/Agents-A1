# analyze_mmpbsa Reference

## Usage


### 2. Scenario Description

Detect MM/PBSA and MM/GBSA outputs inside the MM/PBSA workspace, summarize binding energies, and write CSV/plot/MD tables under `work_dir/results`.

Args:
- `work_dir` (str): Workspace generated by `prepare_protein_md`/`gmx_mmpbsa_propro` containing `mmgbsa/` and `mmpbsa/`.

Return:
- `status` (str): `'success'` or `'error'`.
- `msg` (str): Execution summary or failure message.
- `work_dir` (str): Input workspace path.
- `output_dir` (str): Report directory under `work_dir/results`.
- `detected_mode` (str): One of `'dual'`, `'single_pb'`, `'single_gb'`, or `'none'`.
- `detected_inputs` (Dict[str, str | None]): Detected CSV paths.
- `missing_files` (List[str]): Files that were expected but not found.
- `command` (str): Executed analyzer command.
- `files` (Dict[str, str]): Generated artifact paths inside `output_dir`.
- `reports` (Dict[str, str | None]): Named CSV/PNG/MD artifacts when available.

### 3. How to use tool `analyze_mmpbsa`

```python
client = DrugSDAClient("http://180.184.86.2:32208/mcp")
await client.connect()
response = await client.session.call_tool(
    "analyze_mmpbsa",
    arguments={
        "work_dir": "Protein_MD_02",
    },
)
result = DrugSDAClient.parse_result(response)
await client.disconnect()
```
